# Prepaidly - 即将推出页面

这是一个专业的"即将推出"静态页面，用于 https://prepaidly.io/

## 文件说明

- `index.html` - 主页面文件
- `styles.css` - 样式文件
- `logo.svg` - Logo 文件（SVG 格式，可替换）

## 如何使用

1. 将 `logo.svg` 替换为您的实际 Prepaidly logo
   - 建议使用 SVG 格式以获得最佳显示效果
   - 如果使用 PNG/JPG，请修改 `index.html` 中的文件扩展名

2. 部署到您的服务器
   - 上传所有文件到您的网站根目录
   - 确保 `index.html` 设置为默认首页

## 自定义

您可以轻松自定义：
- **颜色**: 修改 `styles.css` 中的渐变背景色
- **文本**: 修改 `index.html` 中的标题和副标题
- **Logo**: 替换 `logo.svg` 文件

## 浏览器支持

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
